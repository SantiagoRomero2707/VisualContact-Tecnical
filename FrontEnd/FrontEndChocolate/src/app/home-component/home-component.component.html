Reto Chocolate Feast

Introducción

A Little Bobby le encanta el chocolate. Frecuentemente va a su tienda favorita de "5 y 10", Penny Auntie, para comprarlos. Van a tener una promoción en Penny Auntie. Si Bobby guarda suficientes envolturas, puede entregarlas por un chocolate gratis.

Reto

Función Descripción:

Complete la función chocolateFeast en el editor a continuación.
ChocolateFeast tiene el siguiente parámetro(s):

  • int n: La cantidad inicial de dinero de Bobby
  • int c: el costo de una barra de chocolate
  • int m: el número de envolturas que puede entregar para una barra libre

    • Resultado:

    int: el número de chocolates que Bobby puede comer después de aprovechar al máximo la promoción
